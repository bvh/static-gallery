# About this Example

This directory exercises Static Gallery's features:

- **Site configuration** — `site.conf` sets the title, language, and URL
- **Markdown pages** — this page and the home page are rendered from `.md` files
- **Image galleries** — the `photos/` directory is detected as a gallery
- **Static assets** — `robots.txt` is copied through as-is
- **EXIF metadata** — image camera settings and location are displayed when available
